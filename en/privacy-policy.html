Privacy Policy

1. Data Collection and Usage

This plugin does not collect any personally identifiable information (such as name, email address, or phone number). It may, however, use the following data to perform core functions:

  • User settings and preferences: Such as theme or language settings, stored in Chrome's storage for synchronization across devices.
  • Current tab content: Accessed only upon user action to extract text or save webpage information.

2. Requested Permissions

  • activeTab and tabs: Accesses the URL or content of the current tab only upon user action.
  • storage: Stores user settings and notes to enable cross-device synchronization.
  • contextMenus: Adds custom right-click menu items for quick actions.
  • scripting: Injects code into the webpage (e.g., to highlight text) for interactive features.
  • notifications: Sends notifications to inform users of important events or successful actions.

3. Data Security

All data is stored locally on the user’s device or synchronized through Chrome’s storage. The plugin does not transmit any data to remote servers or third parties.

4. User Rights

Users can view, edit, or delete data stored by the plugin at any time. If users no longer wish to use the plugin, they can uninstall it and remove all associated data.

5. Third-Party Services and Sharing

This plugin does not share data with any third parties, nor does it use third-party services to process data.

6. Policy Changes

If this privacy policy is updated, users will be notified through the plugin’s update log. Continued use of the plugin indicates acceptance of the updated policy.
